## 🛠 Project Development & Setup

### Requirements
*   Node.js (v18+)
*   npm

### Installation
1. Navigate to this directory:
   ```bash
   cd frontend/react-portfolio
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Scripts
*   **Development Server:** `npm run dev`
*   **Build Production:** `npm run build`
*   **Linter Analysis:** `npm run lint`
